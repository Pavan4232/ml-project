# 🏥 Disease Prediction System

## 📌 Overview
A Machine Learning project that predicts diseases based on user-selected symptoms using **Random Forest** and **SVM**, with a simple **Streamlit UI**.
---
## 🚀 Features
* Predict disease from symptoms
* Uses Random Forest & SVM
* Simple and interactive UI
---
## ⚙️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
---
## ▶️ Run the Project
```bash
pip install pandas scikit-learn streamlit
streamlit run hospital.py
```
## 📂 Files
* `hospital.py` → main app
* `training.csv` → dataset
---
## 💼 Description
ML-based system that analyzes symptoms and predicts possible diseases quickly and efficiently.
